# Chat Bot Seed


##Command - Response

**/ping** - Pong!

**/about** - Bot name

**/echo** - Echo

**/roll** - random number 1 - 10

**/dolar** - value of MXN to USD

**/europena** - MXN to EUR, HUF to MXN, HUF to EUR

**/wisdom** - Chinese Proverb

**/dict** - Corrects spelling with missed accents automatically.
